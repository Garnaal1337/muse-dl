## Usage

```
muse-dl --help

USAGE: muse-dl [FLAGS] URL

FLAGS:
	--no-bookmarks: Don't add bookmarks
	--no-cleanup: Don't cleanup temporary files
	--tmp-dir /path: Use /path as temporary directory
	--output file.pdf: Use file.pdf as output filename
```
